Trabalhando com fontes em CSS

Instruções

Crie um arquivo HTML e um arquivo CSS para praticar o uso de fontes em CSS. No arquivo HTML, adicione conteúdo que será estilizado com o CSS.

Você deve aplicar diferentes estilos de fonte, como tamanho, família e cor, utilizando as propriedades CSS apropriadas.

As medidas absolutas (cm, mm, in, px, pt, pc) são fixas e não se ajustam ao tamanho da fonte do elemento pai.

Já as medidas relativas (em, rem, %) são mais flexíveis e se adaptam ao tamanho da fonte do elemento pai ou raiz, permitindo que o layout seja mais responsivo.

Experimente também usar fontes personalizadas com a regra @font-face, que permite carregar fontes externas para serem usadas no seu site.

Exemplo de código CSS


body {
  font-family: 'Arial', sans-serif; 
  font-size: 16px; 
  color: #333; 
  line-height: 1.5;
}

h1 {
  font-family: 'Verdana', sans-serif;
  font-size: 2.5em;
  color: #2c3e50;
  text-align: center;
}

h2 {
  font-family: 'Verdana', sans-serif;
  font-size: 1.5em;
  color: #555;
  text-align: center;
}

p {
  font-family: 'Courier New', monospace; 
  font-size: 1em; 
  color: #666; 
  margin: 10px 0;
  text-align: justify;
}

Exemplo de código HTML


<!DOCTYPE html>
<html lang="pt-br">
<head>
  <meta charset="UTF-8">    
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Exercício 2 - Módulo 2</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <header>
    <h1>Trabalhando com fontes em CSS</h1>
  </header> 
  <main>
    <section>
      <h2>Instruções</h2>
      <p>Crie um arquivo HTML e um arquivo CSS...</p>
    </section>
  </main>
</body>
</html>
